# Pass Ambiguity Decision Framework

## Purpose 
Maps orbital element differences to operational leaf nodes. Each branch terminates in an instruction, not a decision.

## Structure 
For each element $\Delta x$:
1. Layer 1 - GS-independent threshold (is separation geometrically meaningful?)
2. Layer 2 - GS-specific threshold (does separation survive observability)
3. Leaf - operational instruction
